# [Doc2] Touchpoint Cadence

**Renewal Cycle (T-60 Days)**
- **T-60**: "Approaching Renewal" - Highlight usage/value.
- **T-45**: "Rate Preview" - If increasing, explain why (inflation, risk).
- **T-30**: "Renewal Offer" - Call to action to confirm details.
- **T-7**: "Urgent: Finalize Renewal".

**Lapse Rescue Cycle (Post Missed Payment)**
- **Day 1**: Automated SMS.
- **Day 3**: Email with "Pay Now" link.
- **Day 7**: Agent Task generated (for high risk).
- **Day 14**: "Grace Period Ending Soon" letter.
